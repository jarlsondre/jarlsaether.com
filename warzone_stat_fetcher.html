Warzone stat fetcher

This is a page for fetching a leaderboard for KG E-Sport. If the table says "placeholder" then your browser rejects the fetch to the server.
In order to fix this you must directly access the server and accept the fetch. After this the leaderboard will load correctly.
You can access the server by typing "https://128.199.56.250:8080" in your browser and then accepting the unsecure connection.
Your browser marks this connection as unsecure because the https-certificate is self-signed.

You can sort by a certain stat by clicking on the corresponding column.
